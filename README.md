# 🎵 Музыкальный сайт - Каркас

Современный и адаптивный сайт для размещения музыки разных жанров.

## 📋 Структура проекта

```
Picotiny/
├── index.html          # Основной HTML файл
├── styles.css          # Стили CSS
├── script.js           # JavaScript функциональность
└── README.md           # Этот файл
```

## ✨ Особенности

- ✅ **Многожанровая музыка** - 6 основных жанров (Поп, Рок, Электроника, Джаз, Хип-Хоп, Акустика)
- ✅ **Встроенный плеер** - Управление воспроизведением, громкость, прогресс-бар
- ✅ **Фильтрация по жанрам** - Быстрый выбор музыки по категориям
- ✅ **Управление плейлистами** - Просмотр всех треков с возможностью выбора
- ✅ **Альбомы** - Галерея альбомов с обложками
- ✅ **Адаптивный дизайн** - Корректно работает на мобильных, планшетах и ПК
- ✅ **Современный интерфейс** - Красивый градиентный дизайн с анимациями
- ✅ **Плавная навигация** - Sticky навигация и smooth scroll

## 🚀 Как начать

1. **Откройте `index.html`** в браузере
2. Используйте **навигацию вверху** для перемещения между разделами
3. **Нажмите на жанр** чтобы отфильтровать музыку
4. **Используйте плеер** для воспроизведения музыки
5. **Кликните на трек** в плейлисте чтобы начать его воспроизведение

## 📝 Как добавить свою музыку

### 1. Отредактируйте массив `tracks` в `script.js`:

```javascript
const tracks = [
    {
        id: 1,
        title: "Название трека",
        artist: "Имя исполнителя",
        genre: "pop", // pop, rock, electronic, jazz, hip-hop, acoustic
        duration: 213, // в секундах
        src: "https://ссылка-на-аудиофайл.mp3",
        album: "Название альбома",
        cover: "https://ссылка-на-обложку.jpg"
    },
    // Добавьте ещё треки...
];
```

### 2. Используйте локальные файлы:

Создайте папки `music/` и `images/`:
```
Picotiny/
├── index.html
├── styles.css
├── script.js
├── music/
│   ├── song1.mp3
│   └── song2.mp3
└── images/
    ├── cover1.jpg
    └── cover2.jpg
```

Затем обновите пути в `script.js`:
```javascript
src: "music/my-song.mp3"
cover: "images/album-cover.jpg"
```
