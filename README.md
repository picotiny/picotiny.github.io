# 🎵 Музыкальный сайт

Персональный сайт для ознакомления с музыкой автора.

## 📋 Структура проекта

```
Picotiny/
├── index.html          # Основной HTML файл
├── styles.css          # Стили CSS
├── script.js           # JavaScript функциональность
└── README.md           # Этот файл
```

## ✨ Особенности

- ✅ **Встроенный плеер** - Управление воспроизведением, громкость, прогресс-бар
- ✅ **Светомузыкальная анимация** - музыкальный ореол вокруг плеера

## 🚀 Как начать

1. **Откройте `index.html`** в браузере
2. **Используйте плеер** для воспроизведения музыки


### 2. Используйте локальные файлы:

Создайте папки `music/` и `images/`:
```
Picotiny/
├── index.html
├── styles.css
├── script.js
├── playlist/
│   ├── song1.mp3
│   └── song2.mp3
└── images/
    ├── cover1.jpg
    └── cover2.jpg
```

Затем обновите пути в `script.js`:
```javascript
src: "music/my-song.mp3"
cover: "images/album-cover.jpg"
```

## 🎛️ Управление плеером

| Кнопка | Действие |
|--------|----------|
| ⏮️ | Предыдущий трек |
| ▶️/⏸️ | Воспроизведение/Пауза |
| ⏭️ | Следующий трек |
//| 🔊 | Регулировка громкости |

## 🌐 Настройка сайта

### Измените название:
```html
<title>Мой музыкальный портал</title>
<div class="logo">🎵 MyMusic</div>
```

### Обновите информацию в разделе "О сайте":
```html
<p>Расскажите о себе и своей музыке...</p>
```

### Добавьте социальные ссылки:
```html
<a href="https://vk.com/your-profile">VK</a>
<a href="https://t.me/your-channel">Telegram</a>
<a href="https://instagram.com/your-profile">Instagram</a>
```
